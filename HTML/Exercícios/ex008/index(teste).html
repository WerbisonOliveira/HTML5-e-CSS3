Formatações de texto

Tag mark - utilizada para marcar um texto ou frase especifica e é possivel mudar a cor através do parâmetro style(CSS)

Tag small - utilizada para deixar o texto pequeno.

Tag del - utlizada para delatar um texto (o texto deve ser desconsiderado)

Tag ins - utilizada para sublinhar(inserir,chamar atenção) um texto

Tag sup - utlizada para deixar um texto sobrescrito. Ex: x2

Tag sub - utilizada para deixar um texto subscrito. Ex: H2O
